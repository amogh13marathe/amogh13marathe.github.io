## Featured Work

### 1. Natural Language Rules Engine + AI Agents — $1.45M Saved
Replaced a brittle hard-coded policy engine with a configurable Natural Language Rules Engine. Layered Microsoft Copilot AI agents to translate plain-English business requirements into SpEL (Spring Expression Language) expressions, then reverse-translated results back to plain English for business reviewer approval before execution.

- Two-step agentic loop: English → SpEL rule → English (human-in-the-loop approval)
- Eliminated engineering dependency for non-technical operators configuring billing decision trees
- 90% reduction in rule-setup errors post-AI integration
- Combined $450K (rules engine) + $1M (self-serve Email/SMS platform) in annual savings

### 2. Subscription Renewals API — $800K ARR
Identified a post-expiration gap in the SMB partner billing lifecycle. Built a 0-to-1 renewal API using a post-expiration pricing model that recovered lapsed subscriptions without marketing spend.

- Discovered pre-expiration outreach triggered a costly claim spike — redesigned the timing model
- Generated $800K new ARR from Home Depot USA at zero marketing cost
- Scaled the model to EU retail markets post-launch
- Stack: REST API, schema validation, ETL pipeline, Informatica / Postgres

### 3. Enterprise Data Ingestion Platform — 40% Faster Partner Onboarding
Scaled the enterprise warranty ingestion platform from 40 to 80+ global retail partners while maintaining <5% API failure rate across 50M+ annual transactions.

- Partners: Walmart, Target, Home Depot, Staples Canada
- Launched Invoice Reporting API enabling programmatic billing reconciliation for EU retailers
- Redesigned API documentation, testing sandbox, and onboarding flow end-to-end
- Schema validation and error-handling frameworks sustaining reliability at scale
